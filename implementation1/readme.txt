There are five unique Python scripts to run.
These are q1_2, q1_3, q1_4, q2_1, and q2_3.

In order to run them you'll need to have Python 3 installed.
You'll also need to install NumPy and MatPlotLib.

    python -m pip install -U pip
    python -m pip install -U matplotlib
    python -m pip install -U numpy

## Running the five Python scripts

The first sections' scripts can be ran with:
    python q1_2.py housing_train.txt housing_test.txt
    python q1_3.py housing_train.txt housing_test.txt
    python q1_4.py housing_train.txt housing_test.txt

The second sections' scripts can be ran with:
    python q2_1.py usps_train.txt usps__test.txt learningrate
    python q2_3.py usps_train.txt usps__test.txt lambdas
